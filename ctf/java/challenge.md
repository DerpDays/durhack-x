# Java Trader 

The "modern" successor to the COBOL system, built as a Java thick client. Analyse thow it communicates with the backend, uncover logic flaws, and take control.

You will need to run the client.jar attached on your local device. When prompted, the "Server" will be the following host: `team{i}-thickapp.durhack.qwerty.technology` - e.g. `team1-thickapp.durhack.qwerty.technology` for team 1.

The port is 9999.
